# Azure AI Foundry Workshop

Welcome to the Azure AI Foundry Workshop! This hands-on workshop will guide you through building intelligent AI applications and agents using Azure AI Foundry's unified development platform.

## üéØ What You'll Learn

By completing this workshop, you will:

- Authenticate and configure Azure AI Foundry in your development environment
- Build your first AI application using GPT-4o models
- Create intelligent AI agents with advanced capabilities
- Implement multi-agent systems for complex workflows
- Apply best practices for production AI applications

## üìö Workshop Contents

This workshop includes a progressive series of Jupyter notebooks designed to take you from setup to advanced AI agent development:

### **Notebooks**

1. **`1-authentication.ipynb`** - Azure Authentication & Setup
   - Configure Azure credentials
   - Connect to Azure AI Foundry services
   - Verify your environment

2. **`2-environment_setup.ipynb`** - Environment Configuration
   - Set up your AI Foundry project
   - Configure model deployments
   - Initialize required resources

3. **`3-quick-start.ipynb`** - Your First AI Application
   - Build a simple AI application
   - Work with GPT-4o models
   - Understand prompt engineering basics

4. **`4-simple-ai-agent.ipynb`** - Building AI Agents
   - Create intelligent AI agents
   - Implement agent capabilities and tools
   - Handle agent-user interactions

5. **`5-multi-ai-agent.ipynb`** - Multi-Agent Systems
   - Design collaborative agent architectures
   - Coordinate multiple agents for complex tasks
   - Implement advanced agent workflows

### Prerequisites

**For All Users:**
- **Azure Subscription** - [Create a free account](https://azure.microsoft.com/free/)
- **Visual Studio Code** - [Download VS Code](https://code.visualstudio.com/)
- **Azure AI Foundry Project** - Set up through [Azure AI Foundry Portal](https://ai.azure.com)

**For DevContainer Setup:**
- Docker Desktop installed and running
- Dev Containers extension for VS Code

**For Local Setup:**
- Python 3.11 or later
- Python and Jupyter extensions for VS Code

### Quick Start After Setup

1. **Configure Azure credentials** in `.env` file
2. **Start with the first notebook**: Open `notebooks/1-authentication.ipynb`
3. **Follow the notebooks in order** for the best learning experience
