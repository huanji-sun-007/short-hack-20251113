# Product Backlog Item: Daily Fruit Prices API

**PBI ID:** PBI-001  
**Title:** Build Daily Fruit Prices API  
**Priority:** Medium  
**Story Points:** 5  

## User Story
**As a** developer  
**I want** an API that returns daily fruit prices  
**So that** applications can display current fruit pricing information  

## Acceptance Criteria
- [ ] Create REST API endpoint `GET /api/fruits/prices`
- [ ] Return JSON response with fruit names and current prices
- [ ] Support common fruits: apple, banana, orange, grape, strawberry
- [ ] Include date and currency in response
- [ ] API should respond within 2 seconds
- [ ] Define error handling with expected HTTP status codes:
  - 400 for bad requests
  - 404 for not found
- [ ] Implement error response schema with fields:
  - `error_code` (string)
  - `message` (string)

## Example Response
```json
{
  "date": "2025-11-05",
  "currency": "USD",
  "fruits": [
    {"name": "apple", "price": 2.99},
    {"name": "banana", "price": 1.49}
  ]
}
```

## Definition of Done
- [ ] API endpoint implemented
- [ ] Basic tests written
- [ ] Documentation created
- [ ] Code reviewed and merged
